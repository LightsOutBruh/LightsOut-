    • Lightsout!

    Project maintained by LightsOutBruh Hosted on GitHub Pages — Theme by mattgraham

    LightsOut! is an android application created by David Luong and Kalin Zaluzec.

    The app communicates to a device we built using a Raspberry Pi 2 in order for you to turn your light switch on or off wirelessly!

    There will be future updates as well as instructions on how to build your own device using a Raspberry Pi 2! Look forward to being lazy!
